# Proyecto: Horarios Académicos UNT
Sistema de gestión y generación automática de horarios — Escuela de Ingeniería de Sistemas, UNT.

## Stack (no cambiar versiones)
Next.js 16.2.6 (App Router) · TypeScript strict · Tailwind CSS 4.3.0 · Shadcn/UI v4 · React Hook Form + Zod · TanStack Table · Supabase (PostgreSQL, Auth, RLS) · PDF-lib + ExcelJS · Vercel
